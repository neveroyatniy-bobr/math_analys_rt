Если последовательность  точек $x_k$ из $\mathbb{R}^n$ ограничена, то существует сходящаяся подпоследовательность $x_{k_m}$ 
Доказательство проведем для $\mathbb{R}^2$ для любого n изменения очевидны.
Пусть $x_k = (\alpha_1^{(k)}, \alpha_2^{(k)});$ так как при всех k  = 1, 2, ... выполняется неравенство $\sqrt{(\alpha_1^{(k)})^2 + (\alpha_2^{(k)})^2} \le C$, то модуль каждой координаты меньше C. Последовательность $\alpha_1^{(k)}$ ограничена, поэтому по [[Теорема Больцано-Вейерштрасса]] существует сходящаяся подпоследовательность $\alpha_1^{(k_l)}$, $\lim\limits_{l \to \infty} \alpha_1^{(k_l)} = \alpha_1^{(0)}$. Далее, аналогично в этой подпоследовательности существует еще одна подпоследовательность  $\alpha_2^{(k_{l_m})}$, $\lim\limits_{m \to \infty} \alpha_2^{(k_{l_m})} = \alpha_2^{(0)}$. Предел  $\lim\limits_{m \to \infty} \alpha_1^{(k_{l_m})} = \alpha_1^{(0)}$ (предел под-подпоследовательности). Поэтому по [[Лемма о покоординатных пределах]] $\lim\limits_{m \to \infty} X_{k_{l_m}} = (\alpha_1^{(0)}, \alpha_2^{(0)})$.
